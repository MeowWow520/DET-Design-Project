# NE555 定时器 使用教程

## 产品概述

NE555 是最经典、应用最广泛的定时器 IC 之一，自 1972 年问世以来一直是电子行业的"标准"元件。只需外接少量电阻电容，即可产生精确的延时或振荡波形。

### 基本参数

| 参数 | 规格 |
|------|------|
| 工作电压 VCC | 4.5V ~ 16V（标准双极型），推荐 5V |
| 输出电流 | 最大 ±200mA（灌电流/拉电流） |
| 输出高电平 | VCC − 1.5V（典型） |
| 输出低电平 | ≈ 0.1V（典型） |
| 最高频率 | 500kHz（典型，可达 2MHz） |
| 功耗 | ~10mA（5V 空载） |
| 封装 | 8-pin DIP / SOP / 微型 SOT-23 |
| 温度范围 | 0°C ~ 70°C（NE555），−40°C ~ 85°C（SE555） |
| 内部结构 | 25 个晶体管 + 2 个二极管 + 16 个电阻 |

> 名称"555"来源于内部由 3 个 **5kΩ** 电阻串联构成的分压网络。

## 引脚定义

### 8-pin DIP 封装引脚图

```
            ┌───── U ─────┐
    GND ── 1│             │8── VCC
    TRIG ── 2│   NE555    │7── DISCH
    OUT  ── 3│             │6── THRES
    RESET── 4│             │5── CONT
            └─────────────┘
```

### 引脚功能表

| 引脚 | 符号 | 类型 | 功能说明 |
|------|------|------|---------|
| 1 | GND | 电源 | 地（0V） |
| 2 | TRIG | 输入 | 触发（Trigger）。当电压低于 **1/3 VCC** 时，内部比较器触发，输出翻转为高电平 |
| 3 | OUT | 输出 | 输出端。可驱动 TTL 电路，最大灌/拉电流 200mA |
| 4 | RESET | 输入 | 复位（**低电平有效**）。接低电平时强制输出为低。不使用时接 VCC |
| 5 | CONT | 输入 | 控制电压（Control Voltage）。可外部改变比较器阈值。不使用时经 10nF 电容接地去噪 |
| 6 | THRES | 输入 | 阈值（Threshold）。当电压高于 **2/3 VCC** 时，内部比较器触发，输出翻转为低电平 |
| 7 | DISCH | 输出 | 放电（Discharge）。内部 NPN 三极管集电极，输出低时对地导通，用于给定时电容放电 |
| 8 | VCC | 电源 | 正电源 4.5V ~ 16V |

## 内部功能框图

```
                   VCC (8)
                     │
         ┌───────────┼───────────┐
         │           │           │
        R(5k)       R(5k)       R(5k)
         │           │           │
         ├── 2/3 VCC │           │
         │           ├── 1/3 VCC │
         │           │           │
    ┌────▼────┐ ┌────▼────┐    GND (1)
    │ 比较器1  │ │ 比较器2  │
    │  (TH)   │ │  (TR)   │
    │  +  ────┼─┤  −     │
    │  −  ────┼─┤  +     │
    │ 2/3VCC  │ │ 1/3VCC  │
    └────┬────┘ └────┬────┘
         │     R     │     S
         │     R     │     S
         └─────┼─────┘
               │
          ┌────▼────┐
          │  R-S FF  │
          │  触发器   │
          │  Q    /Q │
          └──┬────┬──┘
             │    │
             │    └──▶ DISCH (7) ──▶ 内部 NPN 管基极
             │
          ┌──▼──┐
          │ 输出 │──▶ OUT (3)
          │ 缓冲 │
          └─────┘

    CONT (5) ──▶ 改变 2/3 VCC 参考电平
    RESET (4) ──▶ 直接复位触发器 (低有效)
```

### 框图说明

| 模块 | 功能 |
|------|------|
| 3×5kΩ 电阻分压网络 | 产生 1/3 VCC 和 2/3 VCC 两个参考电压，分别供给两个比较器 |
| 比较器 1（TH） | 反相输入接 2/3 VCC，同相输入接 THRES(6)。当 THRES > 2/3 VCC → 输出高 → 复位触发器 |
| 比较器 2（TR） | 同相输入接 1/3 VCC，反相输入接 TRIG(2)。当 TRIG < 1/3 VCC → 输出高 → 置位触发器 |
| R-S 触发器 | 由两个比较器控制。置位→OUT=高, DISCH 关断；复位→OUT=低, DISCH 导通 |
| 输出缓冲 | 推挽输出，可灌/拉 200mA，直接驱动 LED、继电器等 |
| 放电管 | NPN 管，OUT 低时导通 → DISCH(7) 对地导通 → 给定时电容放电 |

## 三种工作模式

### 1. 单稳态模式（Monostable）

产生单次脉冲，脉冲宽度由 RC 时间常数决定。

```
        VCC                   VCC
         │                     │
         R                   │  R(上拉)
         │                   │
         ├──────┬────            └── TRIG(2) ← 触发脉冲（负脉宽）
         │      │
         ├── THRES(6)
         │
         C
         │
        GND
```

**脉宽公式**：t = **1.1 × R × C**

| 参数 | 说明 |
|------|------|
| R | 1kΩ ~ 20MΩ |
| C | 100pF ~ 1000μF |
| t | 输出高电平持续时间（秒） |

### 2. 无稳态模式（Astable / 多谐振荡）★本项目使用

自由振荡，持续输出方波，频率和占空比由 R1、R2、C 决定。

#### 标准无稳态电路

```
        VCC                    VCC
         │                      │
        R1                   ┌── RESET(4)
         │                    │
         ├──── DISCH(7)       │
         │                    │
        R2                 ┌──┴── TRIG(2) 与 THRES(6) 短接在一起
         │                  │
         ├── THRES(6) ──────┘
         │
         C ──┬── CONT(5) ── 10nF ── GND（去耦）
         │
        GND
```

#### 充电与放电

| 阶段 | 路径 | 目标电压 | 时间公式 |
|------|------|---------|---------|
| 充电 | VCC → R1 → R2 → C → GND | 1/3 VCC → 2/3 VCC | t_high = **0.693 × (R1 + R2) × C** |
| 放电 | C → R2 → DISCH(7) → GND | 2/3 VCC → 1/3 VCC | t_low = **0.693 × R2 × C** |

#### 频率和占空比

| 参数 | 公式 |
|------|------|
| 周期 T | T = t_high + t_low = 0.693 × (R1 + 2×R2) × C |
| 频率 f | f = **1 / T** = 1.44 / [(R1 + 2×R2) × C] |
| 占空比 D | D = t_high / T = (R1 + R2) / (R1 + 2×R2) |
| 占空比范围 | **50% ~ 100%**（标准接法永远 >50%，因为 R1+R2 > R2） |

#### 50% 占空比改进电路

如果两电阻相等(R1=R2)，理论占空比 ≈ 66.7%。若要精确 50% 占空比：

```
    方法一：添加二极管
    在 R2 旁并联一个二极管（阳极接 DISCH，阴极接 THRES）
    → 充电只经 R1，放电经 R2 → D = R1 / (R1 + R2)
    
    方法二：OUT 接 R2
    将 R2 一端从 VCC 改接到 OUT(3)，一端接 THRES(6)
    → 充电和放电都经 R2 → D ≈ 50%
```

### 3. 双稳态模式（Bistable）

不使用 RC 网络。TRIG(2) 接地置高，RESET(4) 接地置低。相当于 RS 触发器。

## 本项目中的应用

### 电路设计（无稳态模式）

555 产生约 **1kHz** 的方波时钟，驱动 74LS161 计数器。

```
        +5V                     +5V
         │                       │
         R1 = 1kΩ             ┌── RESET(4)    (不使用时接 VCC)
         │                     │
         ├──── DISCH(7)        │
         │                     │
         R2 = 10kΩ          ┌──┴── TRIG(2) ──┐
         │                   │                │
         ├── THRES(6) ───────┘                │
         │                                    │
         C = 68nF         CONT(5) ── 10nF ───┼── GND
         │                                    │
        GND                              OUT(3) ──▶ 74LS161 CLK
```

### 频率计算验证

| 参数 | 值 | 计算 |
|------|-----|------|
| R1 | 1kΩ | |
| R2 | 10kΩ | |
| C | 68nF | |
| t_high | 0.518ms | 0.693 × (1k + 10k) × 68n = 0.693 × 11k × 68n |
| t_low | 0.471ms | 0.693 × 10k × 68n |
| T | 0.989ms | |
| **f** | **≈ 1.01 kHz** | 1 / 0.989ms |

### 常用频率参考表

| 频率 | R1 | R2 | C | t_high | t_low | 占空比 |
|------|-----|-----|-----|--------|--------|--------|
| 1 Hz | 1kΩ | 680kΩ | 1μF | 472ms | 471ms | ~50% |
| 10 Hz | 1kΩ | 68kΩ | 1μF | 47.8ms | 47.1ms | ~50% |
| 100 Hz | 1kΩ | 6.8kΩ | 1μF | 5.4ms | 4.7ms | ~53% |
| 1 kHz | 1kΩ | 10kΩ | 68nF | 0.52ms | 0.47ms | 52.4% |
| 10 kHz | 1kΩ | 1kΩ | 47nF | 65μs | 33μs | 66.7% |
| 100 kHz | 1kΩ | 100Ω | 68nF | 52μs | 4.7μs | 91.7% |

### 555 输出 → 161 计数器 接线

```
                                        +5V
                                        │
                                       ┌┴┐
    ┌──────┐                          │ R (上拉)
    │ NE555│                          └┬┘
    │      │                           │
    │OUT(3)├───────────────────────────▶ 74LS161 CLK (2)
    │      │                               │
    │  GND │                            RCO(15) ──▶ 可选级联
    └──┬───┘                               │
       │                               Q0 Q1 Q2 (14,13,12) ──▶ 地址线 / 译码器
      GND
```

> **注意**：555 OUT 的输出高电平 ≈ VCC−1.5V = 3.5V（5V 供电时），仍在 TTL 高电平范围内，可以直接驱动 74LS161 的 CLK 输入。

## Proteus 中使用 555

### 查找元件

在 Proteus 元件库中搜索：
- `555` — NE555 标准定时器
- `NE555` — 同上
- `LM555` — 兼容型号
- `7555` — CMOS 低功耗版本

### 仿真注意事项

1. **电源**：VCC 接 +5V（与 74 系列一致），GND 接地
2. **RESET(4)**：不使用时接 VCC，悬空可能导致误触发复位
3. **CONT(5)**：经 10nF 电容接地，抑制高频噪声。Proteus 中可以简化省略
4. **观察波形**：在 OUT(3) 放置虚拟示波器探针，验证频率是否符合预期
5. **元件值**：Proteus 中 R 和 C 的值可以设为精确值（如 68nF），不会像实物那样受标准阻容值限制

### 调试技巧

- 如果仿真波形不动，先检查 RESET 是否接了 VCC
- 用虚拟频率计直接测量 OUT(3)，快速验证频率
- 若输出始终为高或低，检查 RC 回路是否断路或短路

## 注意事项

1. **电源去耦**：VCC(8) 与 GND(1) 之间并联 100nF + 10μF 电容
2. **RESET 不悬空**：引脚 4 不用时必须接 VCC
3. **CONT 去耦**：引脚 5 经 10nF 接地，防止噪声干扰
4. **输出驱动**：虽然 555 最大可输出 200mA，但驱动继电器、电机等感性负载时需加续流二极管保护
5. **频率上限**：实际可靠工作频率建议 ≤ 100kHz（双极型 NE555）或 ≤ 500kHz（CMOS 型 7555）
6. **元件取值**：R1 不应小于 1kΩ（限制放电管电流），R1+R2 不应超过 20MΩ（漏电流影响精度）
7. **占空比限制**：标准接法占空比永远 >50%，若需 ≤50%，需加二极管改进
